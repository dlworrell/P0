# ADR-0001: Repository Rename to AES

## Status

Accepted

## Context

The engineering standards repository was originally created as Engineering_Standard. During early Project Zero planning, the repository was renamed to AES.

## Decision

The repository name is AES.

AES stands for Atarix Engineering Standard.

## Rationale

AES is shorter, clearer, and easier to reference in standards, templates, documents, and dependent repositories.

The name also supports rule identifiers such as AES-000, AES-100, and AES-200.

## Consequences

- Future engineering standard documents should refer to the repository as AES.
- The full formal name remains Atarix Engineering Standard.
- Dependent projects should describe themselves as following AES.
- Older references to Engineering_Standard should be treated as historical.
